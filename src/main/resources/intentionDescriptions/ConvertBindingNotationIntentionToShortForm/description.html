Converts a long-form {fx:…} FXML 2.0 binding expression in an attribute value to the equivalent compact notation.

Examples: 

{fx:Evaluate foo}                    →  $foo
{fx:Observe foo}                     →  ${foo}
{fx:Synchronize foo}                 →  #{foo}
{fx:Evaluate ..foo}                  →  $..foo
{fx:Observe ..foo}                   →  ${..foo}
{fx:Synchronize ..foo}               →  #{..foo}
