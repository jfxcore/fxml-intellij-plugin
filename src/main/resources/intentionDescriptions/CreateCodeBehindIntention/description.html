Creates a code-behind class for an FXML file.

When fx:subclass="com.example.MyView" refers to a class that does not yet exist, this action creates a new sibling file (MyView.java or MyView.kt, detected from the source tree path) in the same directory as the FXML file. The FXML file itself is not modified.

The generated class extends the fxml-compiler-generated base class (MyViewBase) and calls initializeComponent() in the constructor or init block.
