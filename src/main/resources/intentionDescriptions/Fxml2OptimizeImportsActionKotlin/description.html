Optimizes imports in a Kotlin file that contains a @ComponentView-annotated class, preserving imports that are referenced only inside the embedded FXML markup.

IntelliJ's built-in Optimize Imports action bypasses the FXML import optimizer and may remove imports that are required by the embedded markup. This action uses the FXML-aware optimizer instead, so markup-only imports are kept while genuinely unused imports are still removed.

The action is available when the caret is inside the import section of a @ComponentView Kotlin file.
