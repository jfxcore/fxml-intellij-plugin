Converts a compact FXML binding expression in an attribute value to the equivalent long-form {fx:…} markup-extension notation.

Examples: 

$foo             →  {fx:Evaluate foo}
${foo}           →  {fx:Observe foo}
#{foo}           →  {fx:Synchronize foo}
$..foo           →  {fx:Evaluate ..foo}
${..foo}         →  {fx:Observe ..foo}
#{..foo}         →  {fx:Synchronize ..foo}
