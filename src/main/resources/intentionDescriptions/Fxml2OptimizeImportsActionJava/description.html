Optimizes imports in a Java file that contains a @ComponentView-annotated class, preserving imports that are referenced only inside the embedded FXML 2.0 markup.

IntelliJ's built-in Optimize Imports action bypasses the FXML 2.0 import optimizer and may remove imports that are required by the embedded markup. This action uses the FXML 2.0-aware optimizer instead, so markup-only imports are kept while genuinely unused imports are still removed.

The action is available when the caret is inside the import section of a @ComponentView Java file.
