Adds an <?import fqn?> processing instruction for a fully-qualified class name used inline in an FXML 2.0 attribute value.

After adding the import the qualifier can be removed using the Redundant qualifier inspection.

Example:

  Before: eventType="$javafx.scene.input.KeyEvent.KEY_PRESSED"
  After adding import: <?import javafx.scene.input.KeyEvent?>
                       (qualifier becomes redundant)
