Reports invalid literal attribute values in FXML/2 files.

Checks that literal values assigned to FXML attributes are compatible with the property's declared type, following type-coercion rules:

  • Enum constants must match a declared constant of the property's enum type.
  • Color names must be a valid JavaFX Color constant or web-color literal.
  • Other types are validated if a matching static field or {@code @NamedArg}-based constructor parameter can be resolved.
