Reports invalid fx: intrinsic attributes in FXML files.

Checks performed:

  • Unknown intrinsic names (e.g. fx:foo)
  • Root-only intrinsics used on non-root elements (e.g. fx:subclass)
  • Invalid, blank, or duplicate fx:id values
  • Conflicting fx:typeArguments and fx:constant on the same element
  • {fx:Null} assigned to a primitive-typed property
